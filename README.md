🧠 Overview

Contest Watch is designed to solve a simple but critical problem:

“How do I keep track of all coding contests without checking multiple websites daily?”

This tool aggregates contest data and provides a unified interface with filters, countdown timers, and direct links to contests.

✨ Features
📅 Unified Contest Dashboard
View contests from multiple platforms in one place
⏳ Real-time Countdown Timers
Know exactly when a contest starts
🔎 Platform Filters
Filter contests by platform (Codeforces, LeetCode, etc.)
🔗 Direct Contest Links
Jump straight to the contest page
📱 Clean & Responsive UI
Works across desktop and mobile

📦 Installation
# Clone the repository
git clone https://github.com/your-username/contest-watch.git

# Navigate into the project
cd contest-watch

# Install dependencies
npm install

# Run the development server
npm run dev
